# Анализ ТОП-10 тендерных площадок и агрегаторов РФ

**Дата анализа:** 23.12.2025

---

## КЛЮЧЕВЫЕ ИЗМЕНЕНИЯ 2025 ГОДА

### Переход ЕИС на новый API (с 01.01.2025)
- **FTP-сервер ЕИС закрыт** с 01.01.2025
- Данные теперь доступны только через **Сервисы отдачи информации (СОИ)**
- Требуется регистрация + аутентификация через ЕСИА
- Документация: Альбом ТФФ на zakupki.gov.ru

---

## ЧАСТЬ 1: Федеральные ЭТП

### 1. ЕИС / zakupki.gov.ru

**Тип:** Государственная система | **Пользователей:** 2.5+ млн/день | **Первоисточник всех данных**

| Категория | Преимущества | Недостатки |
|-----------|--------------|------------|
| **API** | ✅ **СОИ (Сервисы отдачи информации)** — официальный API с 2025 | Сложная аутентификация ЕСИА, лимиты на запросы, требуется сертификат |
| **База данных** | 100% госзакупок РФ по 44-ФЗ и 223-ФЗ | Только госзакупки, нет коммерческих |
| **Скорость** | Real-time (первоисточник) | — |
| **UI/UX** | Мобильное приложение iOS, поиск, избранное | Нестабильность в пиковые моменты, сложная регистрация |
| **Маркетинг** | Единая точка входа, Госуслуги | Гос. ресурс без продвижения |

---

### 2. Сбербанк-АСТ (sberbank-ast.ru)

**Тип:** Федеральная ЭТП | **Год основания:** 2009 | **Рейтинг RAEX:** 5 место

| Категория | Преимущества | Недостатки |
|-----------|--------------|------------|
| **API** | ❌ **Нет публичного API для разработчиков** | Только интеграция с ЕИС (обязательная для всех ЭТП) |
| **База данных** | Крупнейший объем контрактов среди федеральных ЭТП | Только свои торги |
| **UI/UX** | Веб-интерфейс без ПО, чат-бот Сбер А | Нет мобильного приложения, устаревший дизайн |
| **Маркетинг** | Бренд Сбербанка, SberB2B | Слабое продвижение среди МСП |

**Комиссия:** 1% от НМЦ, не более 5000 руб. (СМП — не более 2000 руб.)

---

### 3. РТС-тендер (rts-tender.ru)

**Тип:** Федеральная ЭТП | **Год основания:** 2010 | **Рейтинг RAEX:** 4 место

| Категория | Преимущества | Недостатки |
|-----------|--------------|------------|
| **API** | ❌ **Нет публичного API для разработчиков** | Только интеграция с ЕИС через ЛК |
| **База данных** | 44-ФЗ, 223-ФЗ, коммерческие | Только свои торги |
| **UI/UX** | ✅ Мобильное приложение iOS/Android, аналитика | Противоречивые отзывы об интерфейсе |
| **Маркетинг** | Обучающие программы, «первая» для новичков | Скрытые платные сервисы |

**Комиссия:** 1% от НМЦ, не более 5000 руб.

---

### 4. Росэлторг / ЕЭТП (roseltorg.ru)

**Тип:** Федеральная ЭТП | **Клиентов:** > 1 000 000 организаций

| Категория | Преимущества | Недостатки |
|-----------|--------------|------------|
| **API** | ❌ **Нет публичного API** (есть интеграция с 1С:УХ для заказчиков) | Интеграция только для размещения, не для получения данных |
| **База данных** | Крупнейшая по количеству пользователей | Только свои торги |
| **UI/UX** | Единый Росэлторг.ID, круглосуточная поддержка | Конфликт старого/нового интерфейса |
| **Маркетинг** | Сильнейший бренд, образовательный контент | Навязывание платного ЭДО |

**Комиссия:** 1% от НМЦ

---

### 5. B2B-Center (b2b-center.ru)

**Тип:** Коммерческая ЭТП | **Год основания:** 2002 | **Рейтинг RAEX:** ТОП-3

| Категория | Преимущества | Недостатки |
|-----------|--------------|------------|
| **API** | ⚠️ **SOAP API** (WSDL) — не REST, требует договор | Устаревший протокол, нет публичной документации |
| **База данных** | 40+ отраслевых площадок, 43 вида процедур | Только коммерческие, нет госзакупок |
| **UI/UX** | ML-рекомендации, множество фильтров | Дизайн «для профессионалов» |
| **Маркетинг** | 20+ лет на рынке, собственный центр ЭЦП | Высокие тарифы (до 240К/год) |

---

### 6. Фабрикант (fabrikant.ru)

**Тип:** Федеральная ЭТП | **Год основания:** 2005 | **Доля рынка:** 6%+

| Категория | Преимущества | Недостатки |
|-----------|--------------|------------|
| **API** | ⚠️ **API для SRM-систем** (требует договор, DEMO-стенд) | Не публичный, только для корпоративных клиентов |
| **База данных** | ЕИС + ЕФРСБ (банкротства) | Только свои торги |
| **UI/UX** | Удобный интерфейс, детальные инструкции | Долгое ожидание техподдержки (30-40 мин) |
| **Маркетинг** | Бесплатное обучение офлайн | Платная регистрация (~2895 руб.), навязывание ЭЦП |

---

### 7. ТЭК-Торг (tektorg.ru)

**Тип:** Федеральная ЭТП | **Год основания:** 2012 | **Объем торгов 2024:** 5.5 трлн руб.

| Категория | Преимущества | Недостатки |
|-----------|--------------|------------|
| **API** | ⚠️ **Интеграция с ERP** (для крупных заказчиков), 80+ рег. систем | Не публичный API |
| **База данных** | 9 секций (Роснефть, РЖД, Газпром Бурение) | Фокус на ТЭК |
| **UI/UX** | Нативный интерфейс, ИИ-сервис «Цифровой помощник» (2025) | Неработающие элементы, нет сортировки |
| **Маркетинг** | Лидер RAEX по малым закупкам | Спам 20 писем/день, техподдержка недоступна |

## НЕДОСТАТКИ СЕРВИСОВ, КОТОРЫЕ МОЖНО ЗАКРЫТЬ НОВЫМ ПРОДУКТОМ

### Таблица недостатков по категориям

| Категория недостатка | Площадки с проблемой | Что можно сделать лучше |
|---------------------|---------------------|------------------------|
| **Нет публичного REST API** | РТС-тендер, Сбербанк-АСТ, Росэлторг, Bicotender, Контур, Синапс, Тендерплан | Предоставить простой REST API с документацией |
| **Сложная аутентификация** | ЕИС (ЕСИА + сертификат) | OAuth 2.0, API-ключи |
| **Лимиты на запросы** | ЕИС, СБИС (200/сутки) | Гибкие лимиты, тарифы по потреблению |
| **Задержка данных** | Контур.Закупки (до суток), агрегаторы | Real-time или near real-time (<5 мин) |
| **Нет мобильного приложения** | Сбербанк-АСТ, Росэлторг, B2B-Center, Фабрикант, ТЭК-Торг, все агрегаторы | PWA или нативное приложение |
| **Устаревший дизайн** | Сбербанк-АСТ, ЕИС | Современный UI/UX |
| **Нет поиска внутри документов** | Большинство ЭТП | Full-text search по PDF/DOCX |
| **Навязывание платных услуг** | Росэлторг (ЭДО), Фабрикант (ЭЦП), РТС-тендер | Прозрачное ценообразование |
| **Плохая техподдержка** | Фабрикант (30-40 мин), ТЭК-Торг (2% дозвон) | Чат-бот + быстрая поддержка |
| **Спам-рассылки** | ТЭК-Торг (20 писем/день) | Настраиваемые уведомления |
| **Нет аналитики конкурентов** | Большинство ЭТП | Встроенная аналитика |
| **Нет прогноза победы** | Все, кроме некоторых агрегаторов | ML-модель прогнозирования |
| **Ограниченный охват площадок** | Тендерплан, Контур (370), Синапс (150) | Агрегация 8000+ источников |
| **Нет данных СНГ** | Большинство агрегаторов | Международные тендеры |

### Ключевые возможности для нового сервиса

| Функция | Текущая ситуация на рынке | Наше решение |
|---------|--------------------------|--------------|
| **REST API** | Есть только у Seldon, СБИС, ЕИС (сложный) | ✅ Простой REST API с Swagger-документацией |

| **Охват источников** | Max: Bicotender (8000), Seldon (7800) | ✅ 8000+ источников |
| **Мобильное приложение** | Только РТС-тендер (ЭТП), ЕИС (iOS) | ✅ PWA + iOS + Android |
| **Аутентификация** | ЕСИА, сертификаты, сложные процедуры | ✅ OAuth 2.0 + API-ключи |
| **Аналитика** | Разрозненная, платная | ✅ Встроенная аналитика конкурентов |
| **Прогноз победы** | Единичные решения | ✅ ML-модель на исторических данных |
| **Поиск по документам** | Почти нигде нет | ✅ Full-text search PDF/DOCX |
| **Ценообразование** | Скрытые комиссии, навязывание | ✅ Прозрачные тарифы, pay-as-you-go |
