# Анализ ТОП-10 тендерных площадок и агрегаторов РФ

**Дата анализа:** 23.12.2025

---

## КЛЮЧЕВЫЕ ИЗМЕНЕНИЯ 2025 ГОДА

### Переход ЕИС на новый API (с 01.01.2025)
- **FTP-сервер ЕИС закрыт** с 01.01.2025
- Данные теперь доступны только через **Сервисы отдачи информации** (СОИ)
- Требуется аутентификация через ЕСИА
- Это затронуло работу ВСЕХ агрегаторов — они должны были адаптироваться к новому API

---

## ЧАСТЬ 1: Федеральные ЭТП

### 1. ЕИС / zakupki.gov.ru

**Тип:** Государственная система | **Пользователей:** 2.5+ млн/день

| Категория | Преимущества | Недостатки |
|-----------|--------------|------------|
| **API** | ✅ Полноценный API через Сервисы отдачи информации (с 2025), интеграция с ERP/1С | Сложная аутентификация через ЕСИА, требуется сертификат |
| **База данных** | 100% госзакупок РФ, первоисточник всех данных по 44-ФЗ и 223-ФЗ | — |
| **UI/UX** | Мобильное приложение iOS, удобный поиск, избранное | Сайт нестабилен в пиковые моменты, сложная регистрация |
| **Маркетинг** | Единая точка входа, интеграция с Госуслугами | Гос. ресурс без продвижения |

**Важно:** ЕИС — первоисточник данных. Все агрегаторы получают данные из ЕИС.

---

### 2. Сбербанк-АСТ (sberbank-ast.ru)

**Тип:** Федеральная ЭТП | **Год основания:** 2009 | **Рейтинг RAEX:** 5 место

| Категория | Преимущества | Недостатки |
|-----------|--------------|------------|
| **API** | ✅ Интеграция с ЕИС через API, автоматическая синхронизация данных на zakupki.gov.ru, экосистема Sber API (400+ сервисов) | Прямой публичный API для сторонних разработчиков ограничен |
| **База данных** | Крупнейший объем контрактов среди федеральных ЭТП | — |
| **UI/UX** | Веб-интерфейс без установки ПО, чат-бот Ассистент Сбер А | Нет мобильного приложения, только русский язык |
| **Маркетинг** | Бренд Сбербанка, SberB2B, стабильная техподдержка | Слабое продвижение среди МСП |

**Комиссия:** 1% от НМЦ, не более 5000 руб. (СМП — не более 2000 руб.)

### 3. РТС-тендер (rts-tender.ru)

**Тип:** Федеральная ЭТП | **Год основания:** 2010 | **Рейтинг RAEX:** 4 место

| Категория | Преимущества | Недостатки |
|-----------|--------------|------------|
| **API** | ✅ Есть API, интеграция с ЕИС настраивается в ЛК | Публичная документация ограничена |

| **Маркетинг** | «Первая» площадка для новичков, обучающие программы | Жалобы на скрытые платные сервисы |

**Комиссия:** 1% от НМЦ, не более 5000 руб. По 223-ФЗ безлимит — 299 000 ₽/год

## РЕКОМЕНДАЦИИ

### По API/Интеграции:
1. **Для работы с ЕИС напрямую** — используйте новые Сервисы отдачи информации (с 2025)
2. **Для интеграции в CRM** — Seldon API или СБИС API
3. **Для ERP/SRM систем** — B2B-Center, Фабрикант, ТЭК-Торг

### По скорости и полноте данных:
1. **Максимальная скорость** — Bicotender (ежесекундно), СБИС (мгновенно)
2. **Максимальный охват** — Bicotender (8000+ источников), Seldon (7800 источников)
3. **Международные тендеры** — Seldon (СНГ), TenderGuru

### По UI/UX:
1. **Для новичков** — РТС-тендер, Контур.Закупки
2. **Для профессионалов** — B2B-Center, Seldon
3. **Для мобильной работы** — РТС-тендер (единственная ЭТП с полноценным приложением)
